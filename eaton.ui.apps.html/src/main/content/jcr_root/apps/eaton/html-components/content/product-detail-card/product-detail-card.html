Core features

Works as a rackmount or tower unit and can be combined to support all types of configurations.LCD display provides 13 measurements, 25 settings, 15 control functionalities, 6 points of identification and optional password protection.Includes remote power off (RPO) and remote on/off (ROO).Online double conversion topology works well in IT, edge networks, industrial automation, healthcare, K-12 and more.

Integrates seamlessly with Eaton's Intelligent Power Manager (IPM) software for continuous uptime of servers and virtual machines. Integrates with Eaton network closet and server room solutions.
